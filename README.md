# AutoQueue + AutoSignLFG

I combined 2 WeakAuras into a standalone addon, since WeakAuras no longer works.

**🚀 WORKS ON MIDNIGHT!**

---

## 🛠 Features

### AutoQueue
* **Automatic Accept:** Automatically clicks the **"Accept"** button when your group leader signs up for an activity.
* **Minimap Button:** Includes a simple minimap icon to toggle the feature on/off, allowing you to switch roles or preferences easily.

### AutoSignLFG
* **Quick Sign-up:** Automatically sign up in LFG with a simple **double-click**.
* **Safe Mode:** To sign up the traditional way, just hold `Shift` + double-click (or `Shift` + click on "Sign Up").
* **Toggleable:** Use `/lfg off` if you want to disable this specific feature.

---

## 💻 Slash Commands

| Command | Action |
| :--- | :--- |
| `/aq on` | Turn on AutoQueue |
| `/aq off` | Turn off AutoQueue |
| `/aq minimap` | Hide or Show the minimap button |
| `/lfg on` | Turn on AutoSignLFG |
| `/lfg off` | Turn off AutoSignLFG |
